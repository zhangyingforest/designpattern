SOLID原则总结
•	单一职责原则 single responsibility 告诉我们实现类要职责单一；
•	而开闭原则 open closed principle 是总纲，他告诉我们要对扩展开放，对修改关闭。
•	里氏替换原则 liskov's substitution principle 告诉我们不要破坏继承体系；
•	接口隔离原则 interface segregation principle 告诉我们在设计接口的时候要精简单一；
•	依赖倒置原则 dependency inversion principle 告诉我们要面向接口编程；
•	迪米特法则 the law of demeter 告诉我们要降低耦合；
